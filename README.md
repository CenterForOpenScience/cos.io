# COS-Site

<h3>Installation</h3>

$ pip install -r requirements.txt

$ python manage.py runserver

go to http://127.0.0.1:8000


<h3>Run as Admin</h3>

$ python manage.py runserver

go to http://127.0.0.1:8000/admin

<h3>Settings</h3>

Go to mysite and change default_local_settings.py to local_settings.py.
